# Vaim-sms
This tool send unlimited messages in any India phone number

## Available On
- Termux
- Kali Linux

## Tested On
- Termux 

## Installation 
- apt update && apt upgrade
- apt install git
- git clone https://github.com/AnonyMux037/Vaim-sms 
- cd Vaim-sms 
- chmod +x *
- bash Vaim-sms.sh
